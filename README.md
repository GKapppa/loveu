# LoveU

LoveU es una aplicación estilo match entre usuarios, inspirada en el funcionamiento de Tinder. El sistema permitirá crear usuarios, perfiles, intereses, preferencias, swipes, matches, mensajes y reportes.

## Tecnologías

- Java 21
- Spring Boot
- Maven
- PostgreSQL
- Spring Data JPA
- Validation
- Lombok
- OpenFeign
